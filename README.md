Sorter for a competitions.

## RU

## ТЗ

## 1. Назначение:

Программа для занесения спортсменов в БД для дальнейшего их распределения в номинациях,
учёта проведённых боёв и их исходов с последующим построением турнирной таблицы.

## 2. Целевая аудитория:

Люди - организаторы и секретари - соревнований(СМБ - Современного Мечевого Боя).

Используется во время соревнований и подготовки к ним.

## 3. Функционал

1. Показывать интерфейс (меню/окна отображения сетки номинации)
2. Отображать меню (menubar/отдельное окно)
3. Показывать интерфейс добавления спортсмена в БД
4. Принимать данные спортсмена
5. Открывать дополнительные окна по нажатию на соответсвующую кнопку меню
6. Иметь постоянную базу данных спортсменов
7. Вносить спортсменов в базу данных
8. Показывать интерфейс отображения сетки номинации
9. Создавать рандомный порядок боёв среди внесённых бойцов
10. Получать результаты боя (в очках и сохранять их до окончания этапа номинации) и продолжать сетку боёв
11. Разделять бойцов на номинации по возрасту, лиге, весу(лёгкий и тяжёлый)
12. Создавать отдельные сетки под разные номинации
13. Отображать меню настроек
14. Изменять настройку минимального кол-ва спортсменов в номинации для разделения
    по весам
15. Возможность проводить повторное взвешивание

### Опционально

1. Показывать интерфейс номинации 4 на 4
2. Перемешать бойцов для вызова в первом раунде
3. Вызывать бойцов по порядку
4. Записывать исход личного боя в очках
5. Показывать интерфейс для записи порядка бойцов для второго раунда(бойцы выходят так как сказали капитаны команд)
6. Вызывать бойцов в порядке заданном лидером
7. Записывать исход боя в очках(хранить до окончания этапа)
8. Создавать сетки боёв команд идентично личным встречам
9. Сохранять исходы боёв
