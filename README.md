# encounter_acounter
Sorter for a competitions.

## RU
### Описание
Программа для занесения спортсменов в БД для дальнейшего их распределения в номинациях,
учёта проведённых боёв и их исходов с последующим построением турнирной таблицы.

### ТЗ
Сделать программу, которая может:
  1. Записывать новых спортсменов на соревнования
  2. Сортировать номинации по врзрасту, лиге, массе(если требуется)
  3. Составлять случайные сетки состязаний
  4. Быть удобной

Опционально:
  1. Создать отдельную систему для командных состязаний(для мечевого боя 4 на 4)
  2. Предоставить удобный доступ ко всей информации
